# Session 2026-04-05 — Диагностика 401, фикс lighttpd auth

## Проблема
Поллер получал 401 Unauthorized при каждом опросе Pi с ~18:28 04.04.
Данные не писались в БД ~14 часов.

## Причина
dahua-auth.conf (проект DahuaAudio) закрывал паролем ВЕСЬ lighttpd сервер
(auth.require на "/"), включая /tar1090/data/aircraft.json.

## Фикс
Изменён /etc/lighttpd/conf-enabled/dahua-auth.conf на Pi:
- Было: auth на "/" (всё)
- Стало: auth только на пути кроме /tar1090/, /data/, /chunks/, /globe_history/

Поллер восстановил работу: batch=1-2, ошибок нет.

## Правило
Если DahuaAudio снова обновляет auth в lighttpd — проверять что /tar1090/ исключён.
