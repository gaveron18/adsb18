# Session 2026-04-05 — Диагностика 401, фикс lighttpd auth

## Проблема
Поллер получал 401 Unauthorized при каждом опросе Pi с ~18:28 04.04.
Данные не писались в БД ~14 часов.

## Причина
dahua-auth.conf (проект DahuaAudio) закрывал паролем ВЕСЬ lighttpd сервер
(auth.require на "/"), включая /tar1090/data/aircraft.json.

## Фикс
Изменён /etc/lighttpd/conf-enabled/dahua-auth.conf на Pi:
- Было: auth на "/" (всё)
- Стало: auth только на пути кроме /tar1090/, /data/, /chunks/, /globe_history/

Поллер восстановил работу: batch=1-2, ошибок нет.

## Правило
Если DahuaAudio снова обновляет auth в lighttpd — проверять что /tar1090/ исключён.

## Фикс: поворот плашки расстояния вдоль линии

Плашка должна быть под тем же углом что и пунктирная линия.

Проблемы которые были:
1. CSS translate перед rotate — translate применялся в повёрнутой системе, плашка смещалась не туда
2. Угол по raw lat/lon — на широте 55N lon/lat имеют разный масштаб в пикселях (cos55 ~0.57)

Исправления:
1. Убрали translate из CSS. Центрирование через Leaflet: iconSize=[120,24], iconAnchor=[60,12]
2. Угол через map.project() — конвертируем lat/lon в пиксели Mercator, atan2 по пикселям
3. text-align: center в .dist-marker
