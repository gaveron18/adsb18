# Session 2026-03-30b — Bug fixes: messageRate, archive GROUP BY, planespotters CORS

## Что сделали

### БАГ 1 — Messages/sec показывал 117877/sec (ИСПРАВЛЕН, коммит 97ee3f2)
- Причина: Pi возвращает messages как накопленный счётчик. tar1090 вычислял дельту, при prev=0 — огромное число.
- Фикс: В server/api/main.py добавили _prev_pi_msgs и _prev_pi_time. Вычисляем messageRate на сервере.
- Результат: 16.0/sec вместо 117877/sec ✅

### БАГ 2 — Archive GROUP BY обрезал треки (ИСПРАВЛЕН, коммит fc4fd1b)
- Причина: GROUP BY icao, callsign + WHERE callsign IS NOT NULL — первые точки без позывного терялись.
- Фикс: GROUP BY icao, MAX(callsign), убрали WHERE callsign IS NOT NULL.
- Результат: NWS743 first_seen на 1.5 мин раньше, +74 точки ✅

### БАГ 3 — Фото бортов planespottersAPI CORS (ИСПРАВЛЕН, коммит f220859)
- Причина: planespottersAPI = true — браузер делал XHR к api.planespotters.net, CORS blocked.
- Фикс: planespottersAPI = false в config.js.
- Проблема кеша: браузер кешировал старый config.js.
- Решение: location = /config.js с Cache-Control no-store в nginx. v=2 в index.html.
- Результат: 0 ошибок в консоли ✅

### Issue 4 — Route колонка пустая (не исправлен)
- tar1090 запрашивает adsb.im/api/routeset — внешний сервис, недоступен с VPS IP.
