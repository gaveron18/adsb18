# TESTING.md — adsb18 archive.html

Ручной чек-лист перед каждым `git push` в `frontend/archive.html`.
Пройти все сценарии. Время: ~10 минут.

**URL:** http://173.249.2.184:8098/archive.html  
**Инструменты:** Открыть DevTools → Console (F12). Ошибки в консоли = провал теста.

---

## Подготовка

1. Открыть архив, выбрать дату с 3+ рейсами
2. Убедиться что список рейсов загрузился (есть строки)
3. Console пустая (нет красных ошибок)

---

## Сценарии

### T1 — Один рейс
1. Поставить галочку на один рейс
2. **Ожидание:** трек появился на карте, счётчик "1/1"
3. Снять галочку
4. **Ожидание:** трек исчез, счётчик "—", карта пустая

### T2 — Несколько рейсов
1. Поставить галочки на 3 разных рейса по одному
2. **Ожидание:** 3 трека на карте, счётчик "3/3"
3. Снять одну галочку
4. **Ожидание:** 2 трека, счётчик "2/2"

### T3 — Кнопка "Все"
1. Нажать кнопку "Все"
2. **Ожидание:** все видимые рейсы загружены, счётчик "N/N"
3. Нет зависших треков (счётчик не "2/5" после окончания загрузки)

### T4 — "Очистить" во время загрузки
1. Нажать "Все" (начнётся загрузка нескольких треков)
2. Сразу нажать "Очистить"
3. **Ожидание:** карта пустая, счётчик "—", нет зависших треков

### T5 — Клик на строку рейса
1. Поставить несколько галочек
2. Кликнуть на строку другого рейса (не на галочку)
3. **Ожидание:** все предыдущие треки сняты, показан только кликнутый

### T6 — Расстояния ON + добавление трека
1. Включить "Расстояния" (кнопка в тулбаре)
2. Поставить точку на карте (клик)
3. Поставить галочку на рейс
4. **Ожидание:** трек появился, расстояние посчиталось, Console без ошибок

### T7 — Расстояния ON + снятие трека
1. Продолжение T6: снять галочку с рейса
2. **Ожидание:** трек исчез, расстояния пересчитались, Console без ошибок

### T8 — Слайдер точек (dot interval)
1. Загрузить 1-2 трека
2. Подвигать слайдер "Точки каждые N км"
3. **Ожидание:** точки на треках перестраиваются, карта не зависает

### T9 — Удаление рейса
1. Загрузить рейс (галочка стоит, трек на карте)
2. Нажать 🗑 на этом рейсе → подтвердить
3. **Ожидание:** рейс исчез из списка, трек исчез с карты

### T10 — Фильтр поиска
1. Напечатать позывной в поле поиска
2. **Ожидание:** список фильтруется, счётчик обновляется
3. Очистить поиск → все рейсы снова видны

## Критерий провала

Любое из:
- Красная ошибка в Console
- Счётчик показывает неверное число после действия
- Галочка снялась сама по себе
- Трек остался на карте после "Очистить"
- Страница зависла
